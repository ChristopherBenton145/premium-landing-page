## Getting Started

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/ChristopherBenton145/premium-landing-page
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the app
   ```js
   npm run dev
   ```
